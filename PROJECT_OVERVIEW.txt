🔬 TECHNICAL SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

ML MODEL:
  Framework: PyTorch 2.0+
  Base Model: EfficientNetB0 (ImageNet pretrained)
  Classes: 2 (Normal/Pneumonia)
  Input: 224×224 RGB images
  Output: Class probability + confidence

INFERENCE:
  Runtime: ONNX Runtime 1.16+
  Target Time: ~150ms per inference (CPU)
  Confidence Calibration: Temperature scaling (T=1.5)
  Batch Support: Dynamic (1 to N)

SUPPORTED FORMATS:
  • PNG - Standard medical imaging
  • JPG - Standard medical imaging
  • DICOM - Medical imaging standard
    - Hounsfield unit normalization (-1024 to 3071)
    - Automatic pixel data extraction
    - Multi-frame support ready

NORMALIZATION:
  ImageNet Mean: [0.485, 0.456, 0.406]
  ImageNet Std:  [0.229, 0.224, 0.225]
  DICOM HU Range: -1024 to 3071 → normalized to 0-1
