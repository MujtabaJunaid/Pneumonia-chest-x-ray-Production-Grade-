📊 DATABASE MODELS (SQLAlchemy)
═══════════════════════════════════════════════════════════════════════════════

TABLE: patients
  id                 INTEGER           PRIMARY KEY
  name               VARCHAR(255)      Required
  age                INTEGER           Required
  gender             VARCHAR(10)       M/F/Other
  contact_info       VARCHAR(255)      Optional
  created_at         DATETIME          Auto-timestamp
  
  Relationships:
    ← predictions (One-to-Many)

TABLE: predictions
  id                 INTEGER           PRIMARY KEY
  patient_id         INTEGER           FOREIGN KEY → patients
  image_path         VARCHAR(512)      File path
  class_index        INTEGER           0 (Normal) or 1 (Pneumonia)
  confidence_score   FLOAT             0-100
  inference_time_ms  INTEGER           Milliseconds
  created_at         DATETIME          Auto-timestamp
  
  Relationships:
    → patient (Many-to-One)

TABLE: audit_logs
  id                 INTEGER           PRIMARY KEY
  endpoint_accessed  VARCHAR(255)      API path
  action_details     TEXT              Full details
  timestamp          DATETIME          Auto-timestamp

🔌 API ENDPOINTS (10 Total)
═══════════════════════════════════════════════════════════════════════════════

PATIENT ROUTES:
  POST   /api/v1/patients/                             Create patient
  GET    /api/v1/patients/                             List all patients
  GET    /api/v1/patients/{patient_id}                 Get patient details
  PUT    /api/v1/patients/{patient_id}                 Update patient

PREDICTION ROUTES:
  POST   /api/v1/predictions/predict-with-patient      ⭐ MAIN ROUTE
  GET    /api/v1/predictions/{prediction_id}           Get prediction
  GET    /api/v1/patients/{patient_id}/predictions     List patient predictions

AUDIT ROUTES:
  GET    /api/v1/audit-logs                            List all audit logs
  GET    /api/v1/audit-logs/patient/{patient_id}       Patient audit logs

HEALTH:
  GET    /health                                       Health check

═══════════════════════════════════════════════════════════════════════════════
⭐ MAIN ENDPOINT: POST /api/v1/predictions/predict-with-patient
═══════════════════════════════════════════════════════════════════════════════

REQUEST:
  Query Parameter: patient_id (integer)
  Form Data: file (UploadFile - PNG/JPG/DICOM)

PROCESS:
  1. ✓ Verify patient exists
  2. ✓ Save X-ray file to disk
  3. ✓ Pass to ML predictor
  4. ✓ Run ONNX inference
  5. ✓ Save prediction to DB
  6. ✓ Create audit log entry
  7. ✓ Return results

RESPONSE (JSON):
  {
    "class_index": 1,                    (0=Normal, 1=Pneumonia)
    "confidence_score": 92.45,           (0-100 percentage)
    "inference_time_ms": 145,            (milliseconds)
    "status_message": "نمونیا کا شبہ",   (URDU message)
    "patient_id": 1,
    "prediction_id": 5
  }

🚀 HOW TO RUN (5 STEPS)
═══════════════════════════════════════════════════════════════════════════════

1. INSTALL DEPENDENCIES
   $ cd backend
   $ pip install -r requirements.txt
   
   Installs: fastapi, uvicorn, sqlalchemy, torch, onnxruntime, pydantic, etc.

2. SETUP ENVIRONMENT (Optional)
   $ cd ..
   $ cp .env.example .env
   # Edit .env if needed (SQLite works out of box)

3. EXPORT ONNX MODEL (if not already done)
   $ python
   >>> from backend.app.ml.ml_pipeline import PneumoniaModel, export_to_onnx
   >>> model = PneumoniaModel()
   >>> export_to_onnx(model)

4. RUN FASTAPI SERVER
   $ uvicorn backend.app.main:app --reload --host 0.0.0.0 --port 8000
   
   Expected output:
   ⚡ ایپلیکیشن شروع ہو رہی ہے...
   📝 ڈیٹابیس ٹیبلز بنائے جا رہے ہیں...
   🤖 ML ماڈل لوڈ کیا جا رہا ہے...
   ✓ ٹیبلز کامیابی سے بنائے گئے
   ✓ ML ماڈل کامیابی سے لوڈ ہوگیا
   
   INFO:     Uvicorn running on http://0.0.0.0:8000

5. ACCESS THE API
   Browser: http://localhost:8000/docs
   Health:  http://localhost:8000/health
   ReDoc:   http://localhost:8000/redoc

💻 EXAMPLE USAGE
═══════════════════════════════════════════════════════════════════════════════

CURL - CREATE PATIENT:
$ curl -X POST "http://localhost:8000/api/v1/patients/" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "احمد علی",
    "age": 45,
    "gender": "M",
    "contact_info": "03001234567"
  }'

CURL - MAKE PREDICTION:
$ curl -X POST "http://localhost:8000/api/v1/predictions/predict-with-patient?patient_id=1" \
  -F "file=@xray.dcm"

PYTHON - MAKE PREDICTION:
import requests

with open("xray.dcm", "rb") as f:
    response = requests.post(
        "http://localhost:8000/api/v1/predictions/predict-with-patient?patient_id=1",
        files={"file": f}
    )
    prediction = response.json()
    print(prediction)
    # {
    #   "class_index": 1,
    #   "confidence_score": 92.45,
